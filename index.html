Sobre mí

Soy David Cereceda, tengo 20 años y soy desarrollador fullstack con experiencia creando aplicaciones web y móviles con tecnologías como Java, JavaScript, Node.js, Express, Python, C#, Flutter y Firebase. Me apasiona construir proyectos estructurados, visuales y eficientes, buscando siempre mejorar cada pequeño detalle del código.

Me desenvuelvo tanto en frontend como en backend, utilizando HTML, CSS y JavaScript para interfaces limpias, y Node.js, Express y MySQL para lógica de servidor y bases de datos. También trabajo con Flutter y Firebase para aplicaciones modernas, rápidas y escalables.

Mi objetivo es crecer como desarrollador, aportar soluciones reales y seguir evolucionando con cada proyecto que construyo.
